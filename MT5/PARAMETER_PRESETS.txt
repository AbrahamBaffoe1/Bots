HOW TO USE PRESETS
================================================================================

1. Choose preset based on:
   - Account size
   - Risk tolerance
   - Market conditions
   - Experience level

2. In MT5:
   - Drag EA to chart
   - Click "Inputs" tab
   - Copy/paste values from preset above
   - Click "OK"

3. Monitor for 1-2 weeks:
   - Track win rate, profit factor
   - Adjust if needed
   - Switch presets if market changes

4. Customize:
   - Start with preset
   - Optimize 1-2 parameters at a time
   - Test changes on demo first

================================================================================
OPTIMIZATION TIPS
================================================================================

If Win Rate < 50%:
→ Increase Min Pattern Confidence (+0.05)
→ Increase Volume Multiplier (+0.2)
→ Increase Min ATR (+0.5)

If Too Few Trades:
→ Decrease Min Pattern Confidence (-0.05)
→ Decrease Min ATR (-0.3)
→ Increase Max Spread Points (+5)

If Drawdown > 20%:
→ Decrease Risk Percent (-0.25%)
→ Increase Stop Loss Pips (+5)
→ Decrease Max Daily Trades (-5)

If Profit Factor < 1.3:
→ Increase Take Profit Pips (+5)
→ Decrease Stop Loss Pips (-3)
→ Better broker (lower spread)
